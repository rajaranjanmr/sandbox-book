
# Sandbox-book

## Tech Stack

**Client:** React, HTML5, CSS3, TailwindCSS, JavaScript

  
## Deployment

Deployed over Sandbox and can be accessed at 
https://rtj5y.csb.app/


## Description

- then it lists 3 books along with rating.
